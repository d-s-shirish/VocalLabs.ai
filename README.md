# Product Teardown Assistant

A single-file web app built for the Vocallabs.ai / Subspace.money Product Intern Assignment (May 2026).

## What it does

Helps you structure your 5 product feedback points in the exact format the assignment requires:

- **(a) Observed** — specific screen, copy, or flow you saw
- **(b) Problem** — real user or business pain
- **(c) Ship instead** — actionable solution

## Features

- Company picker — Vocallabs.ai or Subspace.money
- Framework scratchpads — SWOT, Porter's Five Forces, 7Ps
- 5 feedback cards with pillar tagging (GTM, Competitors, Features, UX)
- Priority levels per feedback
- Live evaluator checklist
- Progress bar
- Export report + copy to clipboard

## Usage

No install needed. Just open the file in any browser:

```bash
open product-teardown-assistant.html
```

Or double-click the file in your file explorer.

## Stack

Plain HTML + CSS + JavaScript. Zero dependencies. No build step.
